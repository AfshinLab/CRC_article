## Steps

### Joint variant calling (Lancet)

The run was done in HPC system, via parallel slurm allocation as in [example of the run code](1.Lancet/parallel_slurm.sh) per chromosome to speed up the calling.


Lancet vcf calles per chromosome were then merged after fixing headers, and the output `fixed_header.vcf` was then filtered to keep only PASS hits, and saved in `P18.onlyPASS_header_fixed.vcf` and `P19.onlyPASS_header_fixed.vcf`. Vcf files were indexed and zipped.


 [code for merge and filter](1.Lancet/1.merge_vcfs.bash) <br> 
 [code for annotation](1.Lancet/2.annot.bash) <br>
 [code for zip and index](1.Lancet/3.zip_index.bash)

maf-files generated by the annotaion step (`P18.onlyPASS.maf`and `P19.onlyPASS.maf`) were then merged after modifiying the headers as in [code for merginng mafs](1.Lancet/4.merge_mafs.bash).
